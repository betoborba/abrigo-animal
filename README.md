# 🐾 Abrigo Animal

Projeto simples em JavaScript para simular o cadastro e gerenciamento de animais em um abrigo.

## 🚀 Funcionalidades

- Cadastro de animais
- Listagem de animais
- Manipulação de listas com JavaScript
- Interação via interface web

## 🛠️ Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## 💡 Objetivo do projeto

Este projeto foi desenvolvido com foco em prática de lógica de programação e manipulação de arrays em JavaScript, simulando um cenário real de cadastro de dados.

## 📂 Estrutura do projeto

- `index.html` → estrutura principal
- `css/style.css` → estilos da aplicação
- `js/script.js` → lógica do sistema
- `assets/` → imagens e recursos visuais

## Como executar

1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador
